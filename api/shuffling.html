Shuffling

concat

template<typename ...Vs>
inline vector<concat_type<Vs...>> kernel_float::concat(const Vs&... inputs)

Concatenate the given vectors into one large vector. For example, given vectors of size 3, size 2 and size 5, this function returns a new vector of size 3+2+5=8. If the vectors are not of the same element type, they will first be cast into a common data type.

Examples

vec<int, 3> x = {1, 2, 3};
int y = 4;
vec<int, 4> z = {5, 6, 7, 8};
vec<int, 8> xyz = concat(x, y, z);  // 1, 2, 3, 4, 5, 6, 7, 8

swizzle

template<size_t... Is, typename V, typename Output = default_storage_type<vector_value_type<V>, sizeof...(Is)>>
inline vector<Output> kernel_float::swizzle(const V &input, index_sequence<Is...> _ = {})

“Swizzles” the vector. Returns a new vector where the elements are provided by the given indices.

Example

vec<int, 6> x = {0, 1, 2, 3, 4, 5, 6};
vec<int, 3> a = swizzle<0, 1, 2>(x);  // 0, 1, 2
vec<int, 3> b = swizzle<2, 1, 0>(x);  // 2, 1, 0
vec<int, 3> c = swizzle<1, 1, 1>(x);  // 1, 1, 1
vec<int, 4> d = swizzle<0, 2, 4, 6>(x);  // 0, 2, 4, 6

first

template<size_t K = 1, typename V, typename Output = default_storage_type<vector_value_type<V>, K>>
inline vector<Output> kernel_float::first(const V &input)

Takes the first N elements from the given vector and returns a new vector of length N.

Example

vec<int, 6> x = {1, 2, 3, 4, 5, 6};
vec<int, 6> y = first<3>(x);  // 1, 2, 3
int z = first(x);  // 1

Takes the last N elements from the given vector and returns a new vector of length N.

Example

vec<int, 6> x = {1, 2, 3, 4, 5, 6};

reversed

template<typename V, typename Output = into_storage_type<V>>
inline vector<Output> kernel_float::reversed(const V &input)

Reverses the elements in the given vector.

Example

vec<int, 6> x = {1, 2, 3, 4, 5, 6};
vec<int, 6> y = reversed(x);  // 6, 5, 4, 3, 2, 1

rotate_left

template<size_t K = 1, typename V, typename Output = into_storage_type<V>>
inline vector<Output> kernel_float::rotate_left(const V &input)

Rotate the given vector K steps to the left. In other words, this move the back element to the front K times. This is the inverse of rotate_right.

Example

vec<int, 6> x = {1, 2, 3, 4, 5, 6};
vec<int, 6> y = rotate_left<4>(x);  // 5, 6, 1, 2, 3, 4

rotate_right

template<size_t K = 1, typename V, typename Output = into_storage_type<V>>
inline vector<Output> kernel_float::rotate_right(const V &input)

Rotate the given vector K steps to the right. In other words, this move the front element to the back K times. This is the inverse of rotate_left.

Example

vec<int, 6> x = {1, 2, 3, 4, 5, 6};
vec<int, 6> y = rotate_right<2>(x);  // 5, 6, 1, 2, 3, 4
